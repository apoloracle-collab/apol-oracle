APOL 3.6 | AI Policy Oracle Labs

"AI won't replace you. Someone using AI will."

APOL 3.6 is an advanced Career Protection Shield designed to analyze professional risks in the era of Artificial Intelligence and Automation. It provides strategic roadmaps for humans to stay ahead of the digital curve.

🔮 Key Features

The Oracle: Dynamic risk assessment for 52+ professions and custom entries.

Survival Metrics: 6-point matrix analysis including Zombie Risk, Abyss Risk, and Cyber-Oracle Fit.

The Visionary Report: AI-generated executive dossiers with actionable market insights.

Global Edition: Multi-language support and international market calibration.

Executive Dossier (PDF): Downloadable professional career protection strategies.

🛠️ Technology Stack

Engine: Python 3.9+

Interface: Streamlit

Intelligence: Google Gemini 1.5 & 2.0 Flash

Visualization: Plotly & Graph Objects

Document Generation: ReportLab

🚀 Deployment & Local Setup

Cloud Deployment

This project is optimized for Streamlit Cloud. Ensure you set your GEMINI_API_KEYS in the Secrets management panel.

Local Installation

Clone the repository: git clone https://github.com/yourusername/apol-oracle.git

Install dependencies: pip install -r requirements.txt

Run the app: streamlit run app.py
